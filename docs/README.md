## About Linkbridge

Linkbridge is a modern, professional educational communication platform designed to replace fragmented communication tools like WhatsApp for educational institutions. It provides a centralized hub for managing assignments, announcements, study materials, events, notices, and real-time communication between students and administrators.

## Key Technologies

- **Frontend**: React.js with Vite
- **Backend**: Node.js with Express.js
- **Database**: MongoDB
- **Real-time**: Socket.IO
- **File Storage**: Cloudinary
- **Authentication**: JWT (JSON Web Tokens)
- **Styling**: Vanilla CSS with Material Design 3 principles
